# 🕵️ The Stationery Detective Quiz

### ✅ Choose the Best Word
1. (Holding an eraser) ➡️ What is (**this / that**)?
2. (Pointing to a ruler far away) ➡️ What is (**this / that**)?

### 🔍 Search & Answer (Wh- Questions)
1. **What** is this? (It is a __________) ✏️
2. **What** is that? (It is an __________) 🧼

### 🏗️ Sentence Builder (Fill in)
1. What ________ that? (is / are)
2. ________ is a long ruler. (It / You)
3. It is ________ orange eraser. (a / an)

---
### 🔑 Teacher Answer Key
* **Choose:** 1. this | 2. that
* **Search:** 1. pencil | 2. eraser
* **Build:** 1. is | 2. It | 3. an
