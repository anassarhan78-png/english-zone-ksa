# 📝 My Meadow Journal

### 🖍️ Task: "Be a Nature Explorer!"
*Look at the meadow. What do you see? Fill in your report.*

---
| 🗺️ Draw your Meadow | 🖋️ Write your Findings |
| :--- | :--- |
|           | 1. I see a __________. (flower / cloud) |
|   [DRAW]  | 2. It is a __________ flower. (red / blue) |
|           | 3. Look! The __________ is in the sky. (sun / bird) |
---

### 💡 Model for Students:
"I see a tree. It is green. I see a bird. It is small and happy."

> **Challenge:** Write the word "Flower" using 3 different colors! 🌈
